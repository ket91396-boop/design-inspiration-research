# Artifact Patterns

Load this when choosing how to analyze references and write prompts for a specific artifact type.

## Logo / Identity

- Look for: mark geometry, negative space, wordmark rhythm, badge structure, reduction to one color, brand-system extensibility.
- Avoid: mascot/detail overload, mockup-only shots, marks too close to existing brands, generated pseudo-lettering.
- Prompt bias: `simple vector logo`, `symbol mark`, `wordmark`, `flat color`, `scalable icon`, `no mockup`.

## Poster / Cover / Key Visual

- Look for: hierarchy, dominant visual zone, type placement, blank space, campaign system, print texture, motion or depth cues.
- Avoid: template layouts, unreadable type, overfilled compositions, copied hero illustrations.
- Prompt bias: make two prompts per direction: one full layout exploration and one no-text background/base image.

## Typography / Title Lettering

- Look for: stroke contrast, character rhythm, ligatures, baseline behavior, cultural reference, real font pairing.
- Avoid: AI gibberish as final type, copied custom lettering, overly decorative characters that harm readability.
- Prompt bias: treat AI as form exploration; final characters should be manually typeset or corrected.

## Packaging

- Look for: shelf impact, material cues, dieline-friendly structure, label hierarchy, ritual of opening, series logic.
- Avoid: pure render mockups with no graphic idea, generic luxury red-gold, unprintable micro-detail.
- Prompt bias: describe package surface or flat graphic system separately from lifestyle mockup.

## UI / Web

- Look for: information density, navigation model, component patterns, visual hierarchy, state handling, interaction clues.
- Avoid: landing-page hero references for operational tools, decorative cards that do not solve workflow, single-hue UI palettes.
- Prompt bias: ask for actual interface screens/components, not marketing illustrations.

## Texture / Wallpaper / Photography

- Look for: material truth, lighting, grain, usable blank areas, crop, repeatability, color temperature.
- Avoid: text, logos, fake watermarks, stock-photo cliches, compositions too busy for downstream use.
- Prompt bias: `no text`, `clean background`, `seamless texture`, `photography`, `wallpaper`, `usable negative space`.

## Illustration / Graphic System

- Look for: shape grammar, icon family rules, character proportion, pattern repeat, motif simplification.
- Avoid: copyrighted characters, exact scene copying, too many unrelated symbols.
- Prompt bias: specify style, motif, palette, and production use; avoid naming living artists unless the user provides a legal reference frame.
